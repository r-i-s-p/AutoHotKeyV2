# AutoHotkey V2 Scripts Collection

A modular collection of AutoHotkey scripts for enhanced productivity, originally consolidated in a single script but separated for improved clarity and maintainability. The scripts are organized by functionality but they can be combined based on your needs.

## Available Scripts

### Text Correction & Enhancement
- **Autocorrect - Spelling**: Common spelling error corrections
- **Autocorrect - Brevity**: Replaces verbose phrases with concise alternatives
- **Autocorrect - Diction**: Improves word choice and phrasing
- **Autocorrect - Capitalization**: Auto-capitalizes common names and proper nouns
- **Autocorrect - Style**: Enforces consistent writing style conventions
- **Autocorrect - American English**: Converts British spellings to American equivalents
- **Accented Words**: Comprehensive list of accented English words and spelling checks (adapted from community contributions by kunkel321 on AHK forums and public GitHub repositories)

### Keyboard Customization
- **Keyboard Remappings**: Custom key bindings for my workflow

### Text Expansion Shortcuts
- **Symbols Shorthand**: Quick insertion of special symbols
- **Links Shorthand**: Rapid access to frequently used URLs
- **LLM Prompts**: Configurable prompts for language models
- **LaTeX Functions**: For mathematical and scientific notation and typesetting
- **HTML/CSS Formatting**: Commonly used HTML tags and CSS properties
- **Python Functions**: Code snippets and function templates

### Utility Functions
- **Google Search Hotkey**: Instant web search from selected text
- **Paywall Bypass**: Navigate around article restrictions
- **Text Editing Tools**: Advanced copy-paste text manipulation
- **Text Case Converter GUI**: Interactive tool for changing text case
- **Edit or Restart Script**: Accessible via hotkeys
- **Miscellaneous Hotkeys**: Additional productivity enhancements

## Installation

1. Install [AutoHotkey](https://www.autohotkey.com/)
2. Copy the compiled script, scriptTextReplaceKeyRemap
3. Run desired scripts individually or combine them as needed

If you want to run the script on startup automatically, follow the instructions:

1. Create a shortcut for your script
2. Place it in the Startup folder which is at the path: `C:\Users\[YourUsername]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`
